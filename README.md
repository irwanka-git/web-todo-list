<h1 align="center">Web Todo List (Tecnical Test)</h1>
Using Docker, Golang, React and MySQL

#Compile React
```
cd react && npm run build
```

#Running Docker
```
docker compose build
```

```
docker compose up -d
```

#Running Unit Test ** 
```
docker compose exec api bash
```
```
go test -v
```


#Akses Web Aplikasi 
```
http://localhost:8000
```

#Akses API Aplikasi 
```
http://localhost:8000/api
```
